# 自定义 Material 3 组件

该模块包含 Material 3 组件的自定义版本，以满足官方库尚不支持的特定设计要求。

## 概述

官方的 Material 3 组件功能强大且灵活，但其中一些组件缺少用于深度定制的必要 API。为了克服这些限制，我们复制了这些组件的原始源代码并对其进行了修改以满足我们的需求。

## 我们的方法

- **选择性定制**：我们仅在绝对必要时才定制组件。
- **代码重复**：定制的组件基于官方源代码，只需进行最少的更改即可达到预期效果。
- **未来兼容性**：我们的目标是尽可能接近原始实现，以方便未来的维护。

## 路线图

我们的目标是最终过渡回官方的 Material 3 组件。我们将积极关注库的更新。一旦官方组件提供了所需的定制选项，我们将弃用我们的自定义实现并切换回官方组件。

# Custom Material 3 Components

This module contains customized versions of Material 3 components to meet specific design requirements that are not yet supported by the official library.

## Overview

The official Material 3 components are powerful and flexible, but some of them lack the necessary APIs for deep customization. To overcome these limitations, we have copied the original source code of these components and modified them to fit our needs.

## Our Approach

- **Selective Customization**: We only customize components when absolutely necessary.
- **Code Duplication**: The customized components are based on the official source code, with minimal changes to achieve the desired effect.
- **Future Compatibility**: We aim to stay as close to the original implementation as possible to facilitate future maintenance.

## Roadmap

Our goal is to eventually transition back to the official Material 3 components. We will actively monitor updates to the library. Once the official components provide the required customization options, we will deprecate our custom implementations and switch back to the official ones.
