🎬 Bangumi API Proxy

基于 Cloudflare Workers 的 Bangumi API 代理服务。

使用方式

{METHOD} https://bangumi-proxy.pages.dev/{完整目标URL}

示例

获取条目信息：

GET /https://api.bgm.tv/v0/subjects/1

搜索：

GET /https://api.bgm.tv/v0/search/subjects?keyword=test

支持的域名

  • api.bgm.tv
  • bgm.tv
  • bangumi.tv
  • chii.in
  • next.bgm.tv

支持的方法

GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS
